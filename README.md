# homebrew-s

Personal Homebrew tap for shanberg tools and utilities.

## Setup

```bash
brew tap shanberg/s
```

## Install

After tapping, install formulae by name. Use `--HEAD` for HEAD-only formulae:

```bash
brew install --HEAD shanberg/s/<formula>
```

## Formulae

| Formula | Description |
|---------|-------------|
| [now](Formula/now.rb) | Minimal, terminal-based focus/task tree (TUI + CLI) |
| [project-manager](Formula/project-manager.rb) | CLI for PARA-style project creation with domain-based numbering (`pm`) |

## Updating

```bash
brew update
brew upgrade <formula>
```
